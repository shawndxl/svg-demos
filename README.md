[2、做一个loading](./demo/test2.html)

为什么要用 SVG ？不如举一个最简单，但也最实用的例子。

假如我们需要一个 loading 的图片，这张图片将用在从大到小各种尺寸上，那么图片尺寸太小的话则放大后失真，太大了又占用流量，loading 图标结构很简单，能否像我们用 CSS 设计一个 div 一样设计一张图片，能够放大缩小又不失真？这就是 SVG 能够轻易做到的。
